# 国际超心理学与中国传统文化电子期刊

这是一个可直接部署到 GitHub Pages 的静态电子期刊网站。

## 发布地址

仓库地址：

```text
https://github.com/qinhong919/UFO
```

GitHub Pages 开启后，访问地址通常是：

```text
https://qinhong919.github.io/UFO/
```

## 发布方式

1. 将本文件夹内容提交并推送到 `main` 分支。
2. 在 GitHub 仓库 Settings → Pages 中选择 `Deploy from a branch`。
3. Branch 选择 `main`，目录选择 `/root`。
4. 等待部署完成后访问上面的 Pages 地址。

## 维护方式

- 首页：`index.html`
- 本期目录：`issues.html`
- 文章页面：`articles/`
- 栏目页面：`sections/`
- 全站样式：`assets/css/style.css`
