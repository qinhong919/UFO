使用说明

1. 在 GitHub 创建一个只给 `qinhong919/UFO` 仓库写入 Contents 权限的 Token。

2. 打开本页，填入 Token，先点“测试权限”。

3. 录入文章时，后台会生成 `articles/xxx.html`，并自动套用 HTML 版文章套头。

4. 公告会写入 `data/announcements.json`。

5. 页面管理会直接改仓库文件，保存前应谨慎检查 HTML。

安全说明：不要把 Token 发给别人，不要写进任何网页内容。使用完可以刷新页面清掉当前会话，或到 GitHub 删除该 Token。
