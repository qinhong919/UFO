# 网站页面说明

本文件用于本地维护。修改网站时，优先找到对应页面文件，改完后用浏览器打开本地 HTML 检查。

## 主要入口

- `index.html`：网站首页。负责展示期刊名称、本期重点、重要学者 Brian D. Josephson、本期推荐文章。
- `issues.html`：本期目录。负责列出创刊号 Vol.1 No.1 的全部内容入口。
- `about.html`：关于本刊。负责说明期刊定位、双语性质、研究范围。
- `submission.html`：投稿须知。负责说明双语摘要、栏目范围、评审制度、投稿材料。
- `editorial-board.html`：编辑委员会。负责显示顾问、主编、副主编、编辑、联系方式。
- `contact.html`：联系我们。负责以后放投稿邮箱、联系人、机构地址、合作方式。

## 栏目页

- `sections/methods.html`：学术论文。列出本刊原创研究论文、综述、方法研究。
- `sections/archive.html`：田野与档案。列出 UFO/UAP、UHP/UOP、田野调查和历史资料方向。
- `sections/scholars.html`：重要学者。创刊号优先介绍 Brian D. Josephson；以后可继续加入国际学者、机构和期刊。
- `sections/international.html`：国际动态。用于放国际超心理学、UFO/UAP、意识研究机构和会议动态。
- `sections/tradition.html`：传统与经验。用于放儒释道、唯识、道教、易学、风水、修身养性等内容。
- `sections/debate.html`：对话与争鸣。用于放不同观点、方法论争鸣、评论和圆桌对话。
- `sections/yi-fang-chang.html`：张一方学术成果专题预告。当前不作为创刊号主栏目，后续期数再集中整理。

## 文章页

- `articles/editorial.html`：发刊词。
- `articles/article-001.html`：张一方《世界超心理学的比较研究和中华超心理学特色》。
- `articles/psychokinesis-plant-granules.html`：孔令宏、容志毅《意念力與植物顆粒相互作用的實驗研究》。

## 资源文件

- `assets/css/style.css`：全站样式。改颜色、字号、卡片、背景、导航，都在这里。
- `assets/js/script.js`：全站脚本。目前功能较少，后续需要交互时再扩展。
- `assets/images/logo.svg`：网站左上角 logo。
- `assets/images/article-header.png`：张一方文章套头图。
- `assets/images/article-header-psychokinesis.png`：意念力文章套头图。
- `assets/downloads/`：Word 原文下载文件目录。

## 常见修改方式

- 改导航：每个 HTML 文件顶部都有 `<nav class="nav">...</nav>`，需要全站统一修改。
- 改编委名单：修改 `editorial-board.html`。
- 新增文章：复制一篇 `articles/*.html`，替换标题、摘要、正文、套头图和下载文件链接，然后在 `issues.html` 与 `sections/methods.html` 加入口。
- 改期刊英文名：全站搜索 `International Journal of Parapsychology` 后统一替换，避免只改首页。
- 改绿色更鲜明：调整 `assets/css/style.css` 里的 `--green` 或 `.site-logo` 的 `filter`。

## 当前排序原则

- 顾问和重要国际学者放前面。
- 编辑名单中，海外成员可适当前置。
- 覃宏放在编辑名单最后，保持谦逊表达；联系方式仍保留在“编辑部联系”中。

## 后台管理

- `admin.html`：期刊管理后台。可用 GitHub Token 录入文章、发布公告、直接编辑页面文件。
- `data/articles.json`：文章索引。目录页和学术论文页会读取这里。
- `data/announcements.json`：公告索引。首页公告区会读取这里。

注意：`admin.html` 是静态后台，真正权限来自 GitHub Token，不要把 Token 写入网页或发给别人。
